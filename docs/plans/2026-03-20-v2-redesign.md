# 番茄课表 V2 重构计划 — 统一日历视图与可展开课程卡片

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将当前 4-Tab 分离式日/周/月视图重构为单一统一日历视图，实现周↔月折叠切换、圆点课时指示、可展开课程卡片、月度统计栏，对齐参考应用的交互模式。

**Architecture:** SwiftUI + SwiftData (MVVM). 主界面合并为一个 ScheduleView，内嵌可切换的 WeekStrip / MonthGrid 日历组件。课程列表按时间分组，卡片支持展开/收起动画。Tab 从 4 个精简为 2 个（课表 + 课程）。

**Tech Stack:** Swift 6.2, SwiftUI, SwiftData, iOS 17+, Xcode 26.3

### 1.4 关键交互特征

| 特征 | 实现细节 |
|------|---------|
| **周↔月折叠切换** | 点击 chevron（∨/∧）切换；右上角日历图标快速切换 |
| **圆点指示器** | 每个日期下方显示 1~4 个小圆点（`●`），对应当天 1~4+ 节课 |
| **今天标记** | 当天日期用金色"今"字徽标（黄色圆圈内嵌"今"） |
| **选中高亮** | 选中日期用青色圆形背景，白色文字 |
| **青色渐变** | 日历区域使用 `#4ECDC4` → `#44B39D` 风格的青绿色渐变背景 |
| **统计栏** | 格式：`X月总排课N次，已上M次`；位于日历和课程列表之间 |
| **可展开卡片** | 每个时间段（如 08:00-10:00）是一行，点击展开显示详细课程信息 |
| **课程进度** | 显示已完成/总计：`4.0/10.0h`（小时数）、`1/8次`（节次） |
| **时间组右侧** | 显示课程序号如 `1/48次` |

## 4. 数据模型详细设计

### 4.1 Course 模型变更

```swift
@Model
final class Course {
    // 已有字段
    var id: UUID
    var name: String
    var colorHex: String
    var notes: String
    var createdAt: Date
    @Relationship(deleteRule: .cascade, inverse: \Lesson.course)
    var lessons: [Lesson]

    // 新增字段
    var subject: String          // 科目/类型（如"阅读"、"全科"、"数学"），空字符串表示未设置
    var totalHours: Double       // 计划总课时（小时），0 表示未设置
    var totalLessons: Int        // 计划总节数，0 表示未设置

    // 计算属性
    var completedLessons: [Lesson] {
        lessons.filter { $0.isCompleted }
    }

    var completedHours: Double {
        completedLessons.reduce(0.0) { $0 + Double($1.durationMinutes) / 60.0 }
    }

    var progressText: String? {
        // 返回 "4.0/10.0h" 格式，无计划课时则返回 nil
        guard totalHours > 0 else { return nil }
        return String(format: "%.1f/%.1fh", completedHours, totalHours)
    }

    var lessonProgressText: String? {
        // 返回 "已上3/10次" 格式
        guard totalLessons > 0 else { return nil }
        return "\(completedLessons.count)/\(totalLessons)次"
    }
}
```

**迁移策略：** 所有新增字段使用默认值（`subject = ""`、`totalHours = 0`、`totalLessons = 0`），SwiftData 自动处理 lightweight migration，不需要手动 migration plan。

### 4.2 Lesson 模型变更

```swift
@Model
final class Lesson {
    // 已有字段
    var id: UUID
    var course: Course?
    var studentName: String

    // 新增字段

    var isCompleted: Bool        // 是否已完成
    var location: String         // 上课地点，空字符串表示未设置

    // 计算属性（已有）
    var duration: TimeInterval { ... }
    var durationMinutes: Int { ... }

    // 新增计算属性
    var sequenceText: String? {
        // 返回 "1/8次" 格式
        guard lessonNumber > 0,
              let total = course?.totalLessons,
              total > 0 else { return nil }
        return "\(lessonNumber)/\(total)次"

    var timeRangeText: String {
        "\(DateHelper.timeString(startTime))-\(DateHelper.timeString(endTime))"
    }
}
```

**迁移策略：** 同上，所有新增字段使用默认值（`lessonNumber = 0`、`isCompleted = false`、`location = ""`），SwiftData 自动 lightweight migration。

## 5. 视图组件详细设计

### 5.1 DotIndicator（圆点指示器）

**职责：** 根据课时数量渲染 1~4 个水平排列的小圆点。

**输入参数：**
- `count: Int` — 课时数量
- `isSelected: Bool` — 所在日期是否被选中（影响颜色）
- `isOnGreenBackground: Bool` — 是否在青色背景上（影响颜色）

**渲染规则：**
- count = 0 → 不渲染
- count = 1 → `●`
- count = 2 → `● ●`
- count = 3 → `● ● ●`
- count >= 4 → `● ● ● ●`（最多显示 4 个）

**颜色：**
- 青色背景 + 未选中 → 白色半透明
- 青色背景 + 选中 → 白色不透明
- 白色背景 → 深灰色

**尺寸：** 每个圆点直径 4pt，间距 2pt

### 5.2 CalendarHeaderView（日历头部区域）

**职责：** 包含月份导航、星期头、日历网格（周或月模式）的容器视图。整体使用青色渐变背景。

**输入/状态：**
- `@Binding selectedDate: Date` — 选中的日期
- `@Binding displayedMonth: Date` — 当前显示的月份
- `@State isExpanded: Bool` — 是否展开为月视图（默认 false = 周模式）
- `lessonCounts: [Date: Int]` — 传入的每日课时计数字典

**子视图组合：**
```
CalendarHeaderView
├── 月份导航行：＜ 2026年03月 ＞  [📅切换图标]
├── 星期头行：周一 周二 ... 周日
├── if isExpanded:
│   └── MonthGridView（完整月历网格 + 圆点）
│   else:
│   └── WeekStripView（单行周日期 + 圆点）
└── Chevron 按钮（∨ 或 ∧）
```

**视觉规格：**
- 背景渐变：`LinearGradient` from `#56C5B9` to `#4AADA3`
- 所有文字颜色：白色
- 今天标记："今" 字在金色（`#D4A537`）圆圈内，字体 `.caption2`
- 选中标记：青色加深（`#2A8A80`）圆形背景
- 圆角：底部 16pt 圆角

### 5.3 WeekStripView（周模式单行）

**职责：** 显示当前选中日期所在周的 7 天，每天显示星期标签、日期数字、圆点。

**布局：** 7 列等宽 HStack

```
 周一    周二    周三    周四    周五    周六    周日
  16      17      18     [今]     20     [21]     22
                          ●             ●●●    ●●●●
```

**每个日期 cell：**
- 星期标签（`.caption2`，白色半透明）
- 日期数字（`.callout`，白色）
- 今天 → 显示"今"替代日期数字，金色圆圈背景
- 选中 → 深色圆形背景
- 下方 DotIndicator

### 5.4 MonthGridView（月模式网格）

**职责：** 显示完整月历，7 列 N 行。每个日期显示数字和圆点。

**布局：** LazyVGrid 7 列

**特殊处理：**
- 上月尾部日期：不显示（留空）
- 下月头部日期：不显示（留空）
- 有课日期以更亮的白色显示
- 周六/周日无特殊颜色区分

### 5.5 StatisticsBar（统计栏）

**职责：** 显示当月排课统计。

**输入：**
- `month: Date` — 当前月份
- `allLessons: [Lesson]` — 所有课时数据

**显示格式：** `X月总排课N次，已上M次`

**视觉：** 居中文字，`font(.subheadline)`，灰色，上下 padding 8pt

### 5.6 LessonTimeGroup（时间分组 — 可展开）

**职责：** 按相同时间段分组显示课时，点击头部切换展开/收起。

**结构：**
```

展开状态：
  🟡 08:00-10:00                        1/48次
```

**头部行：**
- 左侧：橙色时钟图标圆圈 + 时间范围（`08:00-10:00`）
- 右侧：课程序号（`1/48次`）

**交互：** 点击头部 → withAnimation 切换 `isExpanded` 状态

**内部：** 展开时显示 LessonDetailCard

## 7. 风险与注意事项

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| SwiftData migration 失败 | 应用启动崩溃 | 所有新字段使用默认值，确保 lightweight migration 可行；如失败则删除旧数据（MVP 阶段可接受） |
| 日历周/月切换动画卡顿 | 用户体验差 | 使用 `withAnimation(.easeInOut(duration: 0.3))` 而非弹簧动画；月历使用 LazyVGrid 避免一次性渲染 |
| 月历每日圆点计算性能 | 大数据量时日历滑动卡顿 | 预计算 `lessonCountsByDate` 字典，仅在 `allLessons` 或 `displayedMonth` 变化时重算 |
| chevron 交互与 ScrollView 手势冲突 | 展开收起不流畅 | chevron 使用明确的 Button，不依赖手势识别 |
| 展开卡片内容过多超出屏幕 | 下方内容被截断 | 课程列表区域使用 ScrollView 包裹 |
