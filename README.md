# 番茄课表 TomatoSchedule

一款面向教师的 iOS 课程排课管理应用，帮助老师轻松管理课程与课时安排。

## 功能

- 课程管理：创建、编辑课程，支持自定义颜色标识
- 课时排课：为课程添加课时，记录学生、时间、备注等信息
- 多维度日历视图：支持月视图、周视图、日视图查看课程安排
- 数据持久化：基于 SwiftData，数据自动保存在本地

## 技术栈

- Swift 6.0
- SwiftUI
- SwiftData
- iOS 17.0+
- XcodeGen（项目配置管理）

## 运行

1. 安装 [XcodeGen](https://github.com/yonaskolb/XcodeGen)
2. 在项目根目录执行 `xcodegen generate`
3. 使用 Xcode 打开 `TomatoSchedule.xcodeproj`
4. 选择模拟器或真机，运行项目
